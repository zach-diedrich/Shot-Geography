Basketball Shot Distribution Analysis

Project Overview

The analysis presents multiple scenarios with different shot distribution strategies while maintaining the same total number of shots and shooting percentages from each zone. The results demonstrate how reducing mid-range shots in favor of more efficient shooting zones could increase team scoring without increasing the # of possessions per game .

VBA Code Analysis

Shot Distribution Simulation

This simple VBA module assisted me in calculating various realistic scenarios while maintaining shooting percentages to analyze potential scoring improvements.

Sub WhatIfShotDistribution()
    ' Variables for tracking shot data
    Dim targetws As Worksheet
    Dim wsName As String
    Dim ws As Worksheet
    
    Dim ctRimShots As Long
    Dim ctPaintShots As Long
    Dim ctMidRangeShots As Long
    Dim ctBreak3Shots As Long
    Dim ctcorner3Shots As Long
    
    Dim ctCombinedShots As Long
    Dim TotalShots As Long
    
    Dim ctRimMakes As Long
    Dim ctPaintMakes As Long
    Dim ctMidRangeMakes As Long
    Dim ctBreak3Makes As Long
    Dim ctcorner3Makes As Long
    
    Dim ctTotalshotsMade
    
    Dim targetcombinedPCT
    Dim targetCombinedShots As Long
    
    Dim targetRimShots As Long
    Dim targetPaintShots As Long
    Dim targetMidRangeShots As Long
    Dim targetBreak3Shots As Long
    Dim targetcorner3Shots As Long
    
    Dim targetTotal As Double
    
    Dim targetRimMakes As Long
    Dim targetPaintMakes As Long
    Dim targetMidRangeMakes As Long
    Dim targetBreak3Makes As Long
    Dim targetcorner3Makes As Long
    Dim totalRemainingShots As Long
    Dim removedShots As Long
    
    ' Get current shot distribution from the actual data worksheet
    Set ws = ThisWorkbook.Sheets("Actual")
    
    ctRimShots = ws.Range("B2").Value
    ctPaintShots = ws.Range("B3").Value
    ctMidRangeShots = ws.Range("B4").Value
    ctBreak3Shots = ws.Range("B6").Value
    ctcorner3Shots = ws.Range("B7").Value
    ctCombinedShots = ctPaintShots + ctMidRangeShots
    TotalShots = ws.Range("B10").Value
     
    ctRimMakes = ws.Range("D2").Value
    ctPaintMakes = ws.Range("D3").Value
    ctMidRangeMakes = ws.Range("D4").Value
    ctBreak3Makes = ws.Range("D6").Value
    ctcorner3Makes = ws.Range("D7").Value
    
    ' Get user input for target mid-range shot percentage
    On Error Resume Next
    targetcombinedPCT = CDbl(InputBox("Enter the new combined % of shots for midrange and paint (e.g., 0.2 for 20%)"))
    If Err.Number <> 0 Or targetcombinedPCT < 0 Or targetcombinedPCT > 1 Then
        MsgBox "Please enter a valid percentage (between 0 and 1).", vbExclamation
        Exit Sub
    End If
    On Error GoTo 0
    
    ' Set target worksheet for adjusted data
    Set targetws = ThisWorkbook.Sheets("Adjusted")
  
    ' Calculate new shot distribution
    targetCombinedShots = Application.WorksheetFunction.Round(TotalShots * targetcombinedPCT, 0)
    
    removedShots = ctCombinedShots - targetCombinedShots
    
    targetPaintShots = Application.WorksheetFunction.Round(ctPaintShots * (targetCombinedShots / ctCombinedShots), 0)
    targetMidRangeShots = targetCombinedShots - targetPaintShots
    
    ' Redistribute the removed shots proportionally to other shot types
    totalRemainingShots = ctRimShots + ctcorner3Shots + ctBreak3Shots
    targetRimShots = ctRimShots + Application.WorksheetFunction.Round(removedShots * (ctRimShots / totalRemainingShots), 0)
    targetBreak3Shots = ctBreak3Shots + Application.WorksheetFunction.Round(removedShots * (ctBreak3Shots / totalRemainingShots), 0)
    targetcorner3Shots = ctcorner3Shots + Application.WorksheetFunction.Round(removedShots * (ctcorner3Shots / totalRemainingShots), 0)
    
    ' Update target worksheet with new values
    With targetws
        .Range("B2").Value = targetRimShots
        .Range("B3").Value = targetPaintShots
        .Range("B4").Value = targetMidRangeShots
        .Range("B6").Value = targetBreak3Shots
        .Range("B7").Value = targetcorner3Shots
        
        .Range("C2").Formula = .Range("B2") / TotalShots
        
        .Range("B10").Value = TotalShots
        .Range("A10").Value = "Total Shots"
        .Range("A1:A8").Font.Bold = True
        .Columns("A:I").AutoFit
    End With
End Sub

Key features of this VBA module:

  • Takes user input for desired mid-range shot percentage
  • Maintains total shot volume while redistributing shots
  • Proportionally redistributes shots to maintain shooting style
  • Automatically calculates new point totals based on shooting percentages
